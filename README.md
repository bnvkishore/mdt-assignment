
# Augmented-SCM/Nuxt-FE-Amplify

Mobile Application Coding Assignment

## Run Locally

Clone the project

```bash
  git clone https://github.com/bnvkishore/mdt-assignment.git
```

Go to the project directory

```bash
  cd mdt-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Run coverage

```bash
  npm run test
```
